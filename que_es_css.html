Introducción a CSS

1. ¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para definir la presentación de documentos HTML o XML. Su función principal es separar el contenido de su diseño, permitiendo a los desarrolladores controlar aspectos como colores, fuentes y layout. Esto mejora la experiencia del usuario y facilita el mantenimiento de las páginas web.

2. Formas de aplicar CSS en HTML

CSS en línea (inline)

Se aplica directamente en el atributo style de un elemento HTML.
<p style="color: blue; font-size: 20px;">Texto azul.</p>

CSS en el encabezado (internal)

Se define en la etiqueta <style> dentro de la sección <head>.
<head>
<style>
p { color: green; font-size: 18px; }
</style>
</head>

CSS externo

Se vincula un archivo .css externo con la etiqueta <link> en el encabezado.
<head>
<link rel="stylesheet" href="styles.css">
</head>
